# Contributing to Thamizharivu

Thank you for your interest in contributing to Thamizharivu.

This project aims to build open resources for the Tamil language in artificial intelligence and computing.

## Ways to Contribute

### 1. Tamil Datasets

Help collect or curate datasets such as:

- Tamil literature
- Tamil news
- Tamil conversations
- Tamil translations
- Tamil Wikipedia text
- Public domain Tamil books

Datasets should ideally be:

- properly cleaned
- well structured
- documented

### 2. AI / NLP Tools

Develop tools for:

- Tamil tokenization
- Text normalization
- Transliteration
- Tamil language processing
- Dataset preprocessing

### 3. Documentation

Improve documentation for:

- datasets
- data standards
- project guides
- research references

### 4. Literature Digitization

Help digitize:

- public domain Tamil books
- ancient Tamil literature
- rare Tamil manuscripts

## Dataset Guidelines

When contributing datasets:

- ensure they are legally shareable
- include source references
- document preprocessing steps
- include dataset description

## Pull Request Process

1. Fork the repository
2. Create a new branch
3. Commit your changes
4. Submit a Pull Request

All contributions will be reviewed before merging.

## Community Principles

We value:

- openness
- collaboration
- respect for Tamil heritage
- scientific integrity

Thank you for helping grow Tamil in the digital world.
